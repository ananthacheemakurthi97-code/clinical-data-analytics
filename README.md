# Clinical Data Analytics Portfolio
**Anantha Cheemakurthi, PharmD**

This repository contains samples of data analysis work focused on clinical research, drug safety, and operational reporting.

## 📁 Project 1: SQL for Patient Safety Surveillance
* **Objective:** Identify high-risk adverse events from raw clinical datasets.
* **Skills:** Complex Joins, CTEs, Data Filtering.
* **Insight:** Created a query to flag 'Serious' AEs and map them to concomitant medications.

## 📁 Project 2: Python Data Cleaning (Pandas/NumPy)
* **Objective:** Automate the cleaning of raw lab data for SDTM mapping.
* **Skills:** Dataframes, Lambda functions, handling missing values.

## 📁 Project 3: Clinical Visualization (Tableau/Power BI)
* **Objective:** Patient Profile Dashboards.
* **Tools:** Visualizing vitals, AE onset, and lab trends over time for individual subjects.
